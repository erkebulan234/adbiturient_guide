# Recommendation Engine

## Назначение

Рекомендательная система помогает абитуриентам выбрать образовательные программы на основе их интересов и результатов тестирования.

---

## Входные данные

### Профиль пользователя

- уровень образования;
- интересы;
- сильные стороны.

### Результаты теста

Ответы пользователя.

---

## Этапы работы

### 1. Анализ интересов

Определяются доминирующие направления.

---

### 2. Подсчет баллов

Каждой образовательной программе присваивается рейтинг.

---

### 3. Сортировка

Программы сортируются по убыванию рейтинга.

---

### 4. Генерация объяснений

Пользователь получает не только список программ, но и причины выбора.

---

## Схема

```

User Data
↓
Profile
+
Test Results
↓
Scoring
↓
Ranking
↓
Recommendations

```
