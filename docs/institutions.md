# Примечания

* Стоимость обучения зависит от специальности и языка обучения.
* Мин. баллы ЕНТ меняются ежегодно в зависимости от конкурса.
* Большинство вузов участвуют в государственных грантах.
* Некоторые университеты (KBTU, NU, KIMEP) имеют внутренние требования помимо ЕНТ.
* Для IT-направлений конкурс обычно выше среднего.
* Для колледжей чаще используется конкурс аттестатов, а не ЕНТ.
