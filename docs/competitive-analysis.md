# Анализ платформ поступления и образовательных сервисов Казахстана

## Цель исследования

Провести сравнение существующих платформ для абитуриентов Казахстана и определить уникальность будущей системы.

Основная задача:

* понять слабые стороны существующих решений,
* определить рыночную нишу,
* сформулировать уникальное торговое предложение (УТП).

---

# Анализ конкурентов

## 1. MyTest

Преимущества:

* удобный калькулятор поступления,
* анализ шансов по баллам ЕНТ,
* простота интерфейса.

Недостатки:

* отсутствует подбор специальностей по интересам,
* нет колледжей,
* отсутствуют AI-рекомендации,
* система ориентирована только на ЕНТ.

---

## 2. UStudy

Преимущества:

* прогноз проходных баллов,
* анализ грантов,
* исторические данные.

Недостатки:

* отсутствует профориентация,
* нет рекомендаций специальностей,
* нет единой системы сравнения вузов.

---

## 3. Univision

Преимущества:

* большой каталог вузов,
* данные по образовательным программам,
* публикация грантов.

Недостатки:

* нет интеллектуального подбора,
* нет AI,
* отсутствует персонализация.

---

## 4. EduConnect

Преимущества:

* попытка объединить выбор профессии и поступление,
* профориентационные тесты.

Недостатки:

* мало вузов,
* ограниченная база программ,
* отсутствуют колледжи,
* нет полноценной ML-рекомендательной системы.

---

# Главные проблемы рынка

## 1. Разрозненность данных

Сейчас абитуриенту приходится использовать:

* один сайт для ЕНТ,
* другой для грантов,
* третий для вузов,
* четвертый для профориентации.

Нет единой платформы.

---

## 2. Отсутствие персонализации

Большинство платформ:

* показывают только проходные баллы,
* не учитывают интересы пользователя,
* не анализируют склонности,
* не предлагают альтернативные специальности.

---

## 3. Игнорирование колледжей

Почти все сервисы ориентированы только на университеты.

При этом:

* колледжи популярны после 9 класса,
* многие поступают через ТиПО,
* существует сокращенное обучение после колледжа.

---

## 4. Нет AI/ML рекомендаций

Существующие платформы используют:

* фильтры,
* статические таблицы,
* простые калькуляторы.

Практически никто не использует:

* машинное обучение,
* рекомендательные алгоритмы,
* поведенческий анализ,
* прогноз вероятности поступления.

# Уникальное торговое предложение (УТП)

## Основное УТП

«Единственная платформа в Казахстане, которая объединяет:

* интересы абитуриента,
* баллы ЕНТ,
* рекомендации специальностей,
* университеты,
* колледжи,
* и AI-анализ поступления
  в одной системе.»

---

# Ключевые преимущества системы

## 1. Умный подбор специальностей

Система анализирует:

* интересы пользователя,
* любимые предметы,
* баллы ЕНТ,
* карьерные предпочтения.

После этого:

* рекомендует подходящие направления,
* предлагает альтернативные варианты,
* показывает вероятность поступления.

---

## 2. AI-рекомендательная система

ML-модель может учитывать:

* прошлые проходные баллы,
* популярность специальностей,
* вероятность получения гранта,
* успешность поступления других студентов.

---

## 3. Университеты + колледжи

Система объединяет:

* бакалавриат,
* ТиПО,
* сокращенные программы,
* альтернативные траектории обучения.

Это делает платформу полезной:

* после 9 класса,
* после 11 класса,
* после колледжа.

---

## 4. Полная экосистема абитуриента

Платформа может включать:

* калькулятор грантов,
* подбор вузов,
* профориентацию,
* карьерные рекомендации,
* аналитику рынка профессий,
* roadmap поступления.

# Почему проект перспективен

## Причины

1. Высокий ежегодный спрос:

   * сотни тысяч абитуриентов.

2. Рынок до сих пор фрагментирован.

3. Нет сильного AI-продукта в образовательной сфере Казахстана.

4. Возможность масштабирования:

   * СНГ,
   * Узбекистан,
   * Кыргызстан.

5. Возможна монетизация:

   * подписка,
   * реклама вузов,
   * premium analytics,
   * карьерные консультации.

---

# Итог

Большинство существующих платформ:

* решают только одну задачу,
* не имеют персонализации,
* не используют AI,
* не покрывают колледжи.

Предлагаемая система может стать:

* центральной платформой поступления Казахстана,
* AI-ассистентом абитуриента,
* единым агрегатором вузов и колледжей.
