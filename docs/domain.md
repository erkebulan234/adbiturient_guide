# Система образования Казахстана

## Основные уровни образования

### 1. Среднее образование

Обязательное школьное образование:

* Начальная школа
* Основная школа
* Старшая школа (11–12 классы)

После окончания школы выпускник может:

* поступить в университет (бакалавриат),
* поступить в колледж (ТиПО),
* либо начать работать.

---

# ЕНТ (Единое Национальное Тестирование)

ЕНТ - стандартизированный экзамен в Казахстане, используемый:

* для поступления в университеты,
* участия в конкурсе на образовательные гранты,
* оценки знаний выпускников.

## Когда проводится ЕНТ

ЕНТ проходит несколько раз в год:

* Январь
* Март
* Май–июль (основное ЕНТ)
* Август

Для участия в конкурсе на грант учитывается результат основного ЕНТ (май–июль). В основной период можно сдавать ЕНТ дважды и использовать лучший результат.

---

# Структура ЕНТ

## Предметы

ЕНТ включает:

* 3 обязательных предмета:

  * История Казахстана
  * Грамотность чтения
  * Математическая грамотность

* 2 профильных предмета:

  * выбираются в зависимости от специальности

Например:

| Специальность | Профильные предметы              |
| ------------- | -------------------------------- |
| IT            | Математика + Информатика         |
| Медицина      | Биология + Химия                 |
| Юриспруденция | Всемирная история + Основы права |
| Архитектура   | Математика + География           |

---

# Баллы ЕНТ

## Максимальный балл

Максимальный результат - 140 баллов.

## Пороговые баллы

Минимальный балл зависит от направления:

| Направление       | Минимальный балл |
| ----------------- | ---------------- |
| Национальные вузы | 65               |
| Большинство вузов | 50               |
| Педагогика        | 75               |
| Право             | 75               |
| Медицина          | 70               |

# Образовательные гранты

## Что такое грант

Образовательный грант - государственная оплата обучения студента.

Если студент выигрывает грант:

* обучение оплачивает государство,
* студент учится бесплатно,
* иногда выплачивается стипендия.

---

## Как получить грант

Для участия в конкурсе нужно:

1. Сдать основное ЕНТ
2. Набрать проходной балл
3. Подать заявление на конкурс грантов
4. Указать:

   * университеты,
   * образовательные программы,
   * язык обучения

Распределение грантов зависит от:

* балла ЕНТ,
* количества грантов,
* конкурса на специальность,
* квот и льгот.

# Бакалавриат

## Что это

Бакалавриат - первый уровень высшего образования.

Обычно длится:

* 4 года
* иногда 3 года (сокращенное обучение после колледжа)

После окончания выдается степень:

* Bachelor / Бакалавр

---

## Особенности бакалавриата

* академическое высшее образование
* больше теории и фундаментальных знаний
* подготовка:

  * инженеров,
  * программистов,
  * экономистов,
  * врачей,
  * юристов и т.д.

---

# ТиПО (Техническое и профессиональное образование)

ТиПО - система колледжей Казахстана.

## Основные особенности

Обучение:

* после 9 класса
* либо после 11 класса

Длительность:

* 2–4 года

После окончания выдается:

* диплом колледжа,
* квалификация специалиста.

---

## Чем ТиПО отличается от бакалавриата

| Параметр             | ТиПО (Колледж)        | Бакалавриат                   |
| -------------------- | --------------------- | ----------------------------- |
| Тип образования      | Профессиональное      | Высшее                        |
| Длительность         | 2–4 года              | 4 года                        |
| Практика             | Много практики        | Больше теории                 |
| После 9 класса       | Да                    | Нет                           |
| Степень              | Нет                   | Бакалавр                      |
| Возможность работать | Быстро                | Обычно после полного обучения |
| Продолжение обучения | Можно поступить в вуз | Магистратура                  |

---

# Сокращенное обучение после колледжа

Выпускники колледжей могут:

* поступить в университет,
* обучаться по сокращенной программе,
* сдавать отдельный формат ЕНТ.

Это возможно, если специальность колледжа связана со специальностью университета.
