# Предметная область

Abiturient Guide — EdTech-платформа для абитуриентов Казахстана.

Основные задачи:

- помощь в выборе профессии;
- помощь в выборе образовательной программы;
- помощь в выборе университета;
- анализ интересов пользователя;
- персональные рекомендации.

---

## Пользователи системы

### Абитуриент

Проходит тестирование и получает рекомендации.

### Администратор

Управляет данными системы.

---

## Основные сущности

- Пользователь;
- Профиль;
- Университет;
- Специальность;
- Образовательная программа;
- Избранное;
- Тест;
- Рекомендация.
