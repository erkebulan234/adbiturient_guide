# Тестирование

## Используемые технологии

- Vitest
- Supertest

---

## Проверяемые компоненты

### Auth

Проверяются:

- регистрация;
- авторизация;
- JWT-токены.

---

### Recommendation Service

Проверяется:

- логика генерации рекомендаций.

---

### Recommender

Проверяется:

- алгоритм ранжирования программ.

---

### Controllers

Проверяются:

- корректность HTTP-ответов;
- обработка ошибок;
- валидация данных.

---

## Типы тестирования

### Unit Tests

Проверка отдельных сервисов.

### Integration Tests

Проверка взаимодействия компонентов.

### API Tests

Проверка REST API.
