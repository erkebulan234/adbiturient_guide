# Безопасность

## Аутентификация

Используется JWT.

## Авторизация

Реализована ролевая модель:

- user;
- admin.

---

## Защита API

Используются:

- Helmet;
- CORS;
- Express Rate Limit;
- Error Handler.

---

## Валидация данных

Используется библиотека Zod.

---

## Логирование

Используется Winston.

---

## Хранение паролей

Пароли хранятся в виде bcrypt-хэшей.
